# **App Name**: KeepCore

## Core Features:

- Media Upload & Cache: Allow users to upload various media files (images, videos, audio) which are securely stored in the PWA's local cache (e.g., IndexedDB or Cache API) for reliable offline access.
- Offline Media Library: Display an organized library of all uploaded and cached media, accessible at any time without an internet connection. Users can browse their collection seamlessly.
- Integrated Media Player: Enable native playback for cached video and audio files, and allow viewing of cached images directly within the application, offering a comprehensive media experience.
- Media Management: Provide functionalities for users to remove unwanted media files from their local cache, helping them manage storage efficiently.
- PWA Installation & Offline Support: Implement a service worker to transform the site into a Progressive Web App, prompting users to install it and ensuring robust offline functionality and fast loading times.

## Style Guidelines:

- Primary color: A sophisticated, vibrant blue (#619FEA) to convey trust, technology, and clarity, making key actions and elements stand out against a dark background.
- Background color: A very dark, subtle blue-grey (#1C1E22) to establish a clean, professional, and energy-efficient aesthetic suitable for extended viewing in a digital vault.
- Accent color: A deep, muted cyan-blue (#2C89B3) used for secondary interactive elements or status indicators, providing harmonious contrast and visual hierarchy.
- All text: 'Inter', a grotesque-style sans-serif, offering a modern, neutral, and highly readable typeface that perfectly suits a data management application and digital interface.
- Use clean, outlined icons for media controls (play, pause, next), file types (image, video, audio), and standard actions (upload, delete, settings) to maintain a modern and minimalistic aesthetic.
- Adopt a responsive, grid-based layout for the media library to optimize viewing across various devices, with clear, organized sections for different media types and a user-friendly navigation structure.
- Implement subtle, smooth transitions for opening media, navigating between views, and showing loading states, providing a polished and engaging user experience without being distracting.
